keydupl

Duplicate the key into value an Object

Install

$ npm install --save keydupl

Usage

var keydupl = require('keydupl');

keydupl(['a', 'b', 'c']);

//=> { a: 'a', b: 'b', c: 'c' }

API

keydupl([inputs])

inputs

Required Type: array

example : ["a", "b", "c"]
